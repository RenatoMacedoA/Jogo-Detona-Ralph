# 🎮 Detona Ralph

Um jogo inspirado no estilo **Whack-a-Mole**, desenvolvido com **HTML, CSS e JavaScript**, onde o jogador precisa ser rápido para clicar no inimigo que aparece aleatoriamente na tela!

---

## 🚀 Funcionalidades

* ⏱️ Contador regressivo (tempo do jogo)
* 🎯 Sistema de pontuação
* 👾 Inimigo aparece em posições aleatórias
* ❌ Perda de vida ao errar o clique
* ❤️ Sistema de vidas
* 💀 Game Over ao perder todas as vidas ou acabar o tempo
* 🔊 Efeitos sonoros ao acertar

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 🎮 Como jogar

1. Clique no inimigo assim que ele aparecer
2. Cada acerto aumenta sua pontuação
3. Se clicar no lugar errado, você perde uma vida
4. O jogo termina quando:

   * O tempo chega a 0 ⏱️
   * Ou suas vidas acabam ❤️

## 🌎 English Version

### 🎮 Detona Ralph

A reflex-based game inspired by **Whack-a-Mole**, built with **HTML, CSS and JavaScript**, where the player must quickly click the enemy that appears randomly on the screen.

---

### 🚀 Features

* ⏱️ Countdown timer
* 🎯 Score system
* 👾 Random enemy appearance
* ❌ Lose life on wrong click
* ❤️ Lives system
* 💀 Game Over when lives reach 0 or time ends
* 🔊 Sound effects on hit

---

### 🎮 How to play

1. Click the enemy when it appears
2. Each hit increases your score
3. Wrong clicks reduce your lives
4. The game ends when:

   * Time reaches 0 ⏱️
   * Or you lose all lives ❤️
